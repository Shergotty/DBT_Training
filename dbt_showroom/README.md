### Local Development Files

This project uses local configuration files that are ignored by Git:

- **`.user.yml`**: Used for local user-specific configuration overrides (e.g. VS Code extension settings).
- **`.user.id`**: A unique identifier generated by dbt IDE extensions (like dbt Power User) to track local sessions. This file is auto-generated and should not be committed.
